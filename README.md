## 程式功能
採用MVP框架<br/>
使用Butterknife註解框架<br/>
結合了firebase的快速登入選項，可使用Facebook與Google登入，並且有smart-lock功能<br/>

## 程式解說
LandingPage為登入頁，提供1.以信箱註冊帳號 2.已擁有帳號的人登入 3.使用Facebook或Google進行快速登入（登入過一次後，透過smart-lock，可以更快速登入）
